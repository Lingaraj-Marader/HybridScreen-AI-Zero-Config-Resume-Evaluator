About This AI Agent

This project is a Hybrid AI Resume Screener built for the Junior AI Research Associate challenge.

1. Mathematical Baseline: It extracts text from PDFs and runs a Scikit-Learn TF-IDF Cosine Similarity algorithm to check exact keyword matches.

2. Semantic Reasoning: It sends the baseline score and text to an LLM (or a local smart fallback engine) to evaluate transferable skills and gaps.

This ensures no AI hallucinations while maintaining deep context understanding.

Add a New Candidate

This dashboard operates as a secure, local CLI-first application. To process a new resume:

  1. Drag and drop the candidate's PDF into the resumes/ folder on your computer.
  2. Open your terminal and run python agent.py
  3. Refresh this page to see the updated live rankings.
